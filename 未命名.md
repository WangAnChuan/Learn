第三章 方法论

3.1 整体框架

针对水面漂浮物检测中存在的背景动态干扰（如光照反光、连续波纹）以及**特定单一类别（如透明塑料瓶）**视觉特征模糊的问题，本文提出了一种物理感知与指标驱动的目标检测框架（Physically-Informed and Metric-Driven Framework）。如图 3 所示，该框架以 YOLO11 为基准架构，并在两个关键维度上进行了创新性重构：

- **物理感知层面（Data-Centric）**：该框架不再采用传统的单模态输入，转而采用异构多模态联合训练策略。除了传统的光强模态（RGB与Mono），本文进一步引入了包含线偏振度（DoLP）、偏振角（AoP）、Pauli分解及差分（DIF）在内的全偏振物理特征。这种高维度的物理信息输入，不仅补充了**单一目标**表面的几何与材质纹理信息，更促使模型通过多流架构学习跨模态的光照不变性特征，从物理源头增强了模型在强反射或低对比度场景下对**特定透明目标**的鲁棒性。
    
- **指标驱动层面（Model-Centric）**：不同于盲目的网络堆叠，本研究提出了一种指标驱动的架构优化策略。研究利用层级特征聚类评估（FCE）作为诊断探针，量化特征空间的流形分布。实验结果发现特征融合阶段（特别是 Layer 19）存在显著的**目标类内方差漂移（High Intra-class Variance of the Target）**，表明模型在此处未能有效捕捉全局上下文以维持**单一类别**的语义一致性。以此为依据，本文将状态空间模型（Mamba）嵌入该瓶颈层级，利用其线性复杂度的全局感受野压缩**同类目标**特征分布，提升了特征的语义紧致度（Semantic Compactness）与判别力。

3.2 物理感知的异构数据策略

传统的水面目标检测算法往往仅依赖 RGB 图像，这使得模型极易受镜面反射（Specular Reflection）与视觉伪装（Visual Camouflage）的干扰。为了克服这一物理瓶颈，本文并未简单地增加网络输入的通道数，而是提出了一种物理域随机化（Physical Domain Randomization）的数据增强策略。

具体而言，本研究将原始传感器数据解译为 $K=6$ 种异构的物理视图，并将其视为独立的训练样本：

1. 强度模态（Intensity Modalities）：包括可见光（RGB）与单色（Mono）图像。这些模态提供了基础的纹理与几何轮廓信息。
    
2. 偏振模态（Polarization Modalities）：包括线偏振度（DoLP）、偏振角（AoP）以及 Pauli 分解分量。由于水面反射光通常具有高偏振特性，而漫反射的目标物体偏振度较低，这些模态能够为目标与背景的分离提供正交的物理线索。
    

形式化地，构建异构数据集 $D = \{ (I_{RGB}, y), (I_{DoLP}, y), \dots \}$，其中 $y$ 表示**单类别 Bottle 的检测标注（边界框与类别），背景特征由未标注水面区域按策略采样获得**。通过在训练流中混合这些视图，本文旨在促使骨干网络忽略模态特定的高频噪声。**该随机化体现为：训练迭代中随机采样不同物理视图作为输入，使同一 Bottle 在不同物理域投影下被等价学习**，从而迫使模型转而聚焦于学习**目标实体**在不同物理域下保持不变的空间结构特征。
#### 3.3 指标驱动诊断：层级特征聚类评估

为了降低深度神经网络设计中的不可解释性，本研究提出了层级特征聚类评估（Hierarchical Feature Clustering Evaluation, FCE）框架。**针对本研究的单类别检测任务，FCE 旨在量化探测特征流形空间中唯一前景（Target / Bottle）与背景（Background / Water）的二元分布质量**，从而为架构改良提供客观依据。

3.3.1 FCE 评估机制

为了量化特征演化行为，本文定义了 FCE 机制。令 $F_l$ 表示第 $l$ 层的特征张量。本研究通过 RoIAlign 基于真值标注（Ground Truth）与背景采样策略，分别提取**目标特征集合** $V_{target}$ 与背景特征集合 $V_{bg}$。采用主成分分析（PCA）将特征投影至低维流形后，定义 Fisher 判别力分数（$S_{fce}$）作为衡量**二元特征可分性**的核心指标：

$$S_{fce}^{(l)} = \frac{\|\boldsymbol{\mu}_{target} - \boldsymbol{\mu}_{bg}\|_2^2}{\text{Tr}(\mathbf{\Sigma}_{target}) + \text{Tr}(\mathbf{\Sigma}_{bg}) + \epsilon}$$

其中，$\boldsymbol{\mu}_{target}$ 与 $\boldsymbol{\mu}_{bg}$ 分别为**目标簇**与背景簇的质心；$\text{Tr}(\mathbf{\Sigma})$ 表示协方差矩阵的迹，度量分布的紧致度。该指标在物理意义上等价于信噪比（SNR）：分子量化了**目标信号与背景噪声的差异强度**，分母量化了特征分布的发散程度。$S_{fce}$ 值越高，表明该层特征具备越强的语义判别力。**在单类别设定下，该指标刻画的是 Bottle 前景簇与水面背景簇的可分性，而非多类别间的可分性。**

3.3.2 基于 FCE 的特征退化诊断

利用 FCE 探针，本研究对基准模型（YOLO11）的关键层级（索引：13, 16, 19, 22）进行了逐层扫描，揭示了其在复杂水面环境下针对**特定瓶体目标**的特征表达瓶颈。

1. - **总体趋势分析**： 如图 1(a) 所示，FCE 评分随网络深度呈现显著的非单调波动。这表明，虽然检测头（Head）能够通过后处理恢复部分识别能力，但在主干网络的深层特征抽象阶段，模型已经丢失了大量关键判别信息，导致**目标流形与背景流形**出现了断裂。
    
- **指标分解与病理分析**： 通过解耦 FCE 指标的分子与分母，本研究定位了具体的病理特征：
    
    - **目标类内发散（High Target Intra-class Divergence）**：如图 1(b) 所示，Layer 19 表现出异常高的类内离散度。这意味着**同一类目标（瓶子）**在不同姿态、光照下的特征在流形空间中极度发散。模型未能将破碎的瓶体局部特征聚类为一个紧凑的语义整体。
        
    - 前景-背景混叠（Target-Background Aliasing）**：如图 1(c) 所示，Layer 13 的**类间分离度**（此处特指目标与背景的距离）处于最低点。这揭示了模型在此层级无法有效抑制与**目标纹理**高度相似的水波纹噪声，导致**单一前景**与背景在特征空间中发生了严重的语义纠缠。
    

3. 特征流形可视化：
    
    图 1(d) 的二维散点图直观验证了上述诊断：基准模型的 Layer 19（红色数据点）明显偏离了高分离度、低类内方差的理想区域，向着高发散、低区分的区域漂移。此外，图 1(e) 显示主成分分析（PCA）的前 $d$ 个主成分累积解释方差均超过 90%，确证了上述流形投影能够真实反映原始高维特征的拓扑结构。
    

诊断结论：FCE 诊断明确指出，Layer 13 至 Layer 19 是特征提取链路中的语义瓶颈。这种局部的特征退化本质上归因于卷积神经网络缺乏长距离依赖（Long-range Dependency）建模能力，无法利用水面的全局上下文一致性来校正局部的反光噪声或聚合发散的物体特征。这一量化结论为在该层级引入具有全局感受野的状态空间模型（Mamba）提供了理论与数据支撑。

3.4 C3k2-Mamba：上下文感知的骨干重构

为了解决 FCE 诊断出的深层特征语义塌陷问题，需要一种能够突破卷积局部感受野限制的机制。虽然 Transformer 能够建立全局依赖，但其二次方计算复杂度难以满足水面无人设备的实时性要求。因此，本研究引入了选择性状态空间模型（Selective State Space Models, Mamba）。

3.4.1 选择性状态空间模型原理

不同于卷积的滑动窗口机制，Mamba 将图像特征展平为一维序列，并利用线性时不变系统（LTI）进行长距离上下文扫描。其核心离散化状态方程定义如下：

$$h_t = \bar{\mathbf{A}} h_{t-1} + \bar{\mathbf{B}} x_t$$

$$y_t = \mathbf{C} h_t$$

其中，$x_t$ 为输入特征序列，$h_t$ 为隐状态，$y_t$ 为输出。通过这种递归形式，Mamba 能够在线性复杂度下将全图的波浪纹理和光照分布编码进隐状态中。对于目标检测而言，模型在判断局部像素属性时能够参考全图的背景一致性信息，从而有效抑制与全局背景相似的局部反光噪声。

3.4.2 针对性的深层骨干重构

基于 FCE 的诊断结论，本文实施了指标驱动的局部手术策略。为了平衡计算效率与物理感知能力，本研究保留了浅层（P1-P3）的卷积结构，以维持对高频物理特征的提取优势。针对出现严重语义混叠的深层阶段（P4, P5），本文设计并植入了 C3k2-Mamba 模块，以替代原有的 Bottleneck 结构。

C3k2-Mamba 模块设计：

如图 3 所示，该模块将状态空间模型（SSM）嵌入 CSP 流形架构中。本研究采用了一种双流密集聚合（Dual-stream Dense Aggregation）机制：

1. 特征分流：输入张量 $X$ 首先经过 $1\times1$ 卷积降维，并在通道维度被分割为守恒流 $X_{id}$ 和计算流 $X_{comp}$，以确保梯度的无损传播。
    
2. 级联状态空间建模：计算流进入堆叠的 Mamba Blocks 序列。为了提高深层网络的特征复用率，本研究构建了密集连接链路。令 $M_i$ 表示第 $i$ 个 Mamba Block，输出序列 $Y_i$ 递归定义为：
    
    $$Y_0 = X_{comp}, \quad Y_i = M_i(Y_{i-1}), \quad i=1, \dots, N$$
    
3. 全维上下文重组：最终的输出由所有中间状态与初始分流聚合而成。这种设计促使模型在每一层将 Mamba 提取的全局上下文与局部特征进行对齐：
    
    $$Y_{out} = \text{Conv}_{1\times1}\left( \text{Concat}[X_{id}, Y_0, Y_1, \dots, Y_N] \right)$$
    

该模块采用双流密集聚合机制：输入特征被分割为守恒流与计算流。计算流进入堆叠的 Mamba Blocks，通过密集连接（Dense Connection）促进特征复用。最终，通过聚合所有中间状态，模型在每一层将 Mamba 提取的全局上下文与局部特征对齐。这种设计利用全局感受野修复了语义断裂，使模型能将**前景目标**从复杂的背景流形中精确剥离。

第四章 实验与结果分析

4.1 数据集与实验设置

##### 4.1.1 单一类别异构感知基准（Single-Category Heterogeneous Benchmark）

针对水面漂浮物检测中极具挑战性的透明及半透明塑料瓶，本研究构建了一个**专注于单一类别（Single-Category）**的多源异构感知基准。该基准整合了三个互补的数据源，旨在攻克**特定透明瓶体**难识别与动态水面难定位的双重瓶颈。

为了消除语义偏差并锁定研究目标，本研究执行了严格的**标签清洗与映射（Label Mapping）**程序：将源数据中所有相关子类（如 Bottle, Jar, Plastic, Waste）**统一映射为唯一的检测类别：Bottle**。

1. **光学物理锚点（PoTATO Subset）**：精选 PoTATO 数据集中包含瓶类目标的子集。引入该数据源旨在利用其高质量偏振图像，促使模型学习透明瓶体特有的偏振光学指纹（如边缘高偏振度），从而在物理层面解决透明目标的视觉隐身问题。
    
2. **动态环境上下文（FloW-Img Subset）**：该子集提供了USV视角下丰富的水波扰动、倒影粘连及半淹没样本，增强模型对复杂背景的适应性。
    
3. **开放世界泛化（Wild-Bottle 2400+）**：自建的高难度数据集，专注于极端强反光与浑浊污水下的**单一瓶体**检测，用于检验模型在非受控环境下的泛化边界。

##### .1.2 物理感知训练策略

**标签统一与二元任务定义**：为了消除数据集间的语义偏差并锁定研究目标，本文将所有目标标注统一映射为单类别索引 $C_{bottle}$，**将多分类问题转化为更加纯粹的目标-背景（Target-Background）二元检测问题**。

**异构模态联合训练**：针对**单类别训练**，本研究实施了动态模态丢弃策略。这种策略构建了一个对抗环境，促使骨干网络解耦**瓶子这一特定类别**的材质与环境特征：在有偏振数据时，模型侧重于学习**透明材质**的物理几何边缘；在仅有 RGB 数据时，模型则侧重于学习**瓶身**在水面的漂浮形态。

4.1.3 实施细节

所有实验均在搭载 Ubuntu 22.04 与单张 NVIDIA RTX 5090 GPU 的工作站上基于 PyTorch 框架完成。本文选用 YOLO11n 作为基准模型并加载 COCO 预训练权重进行迁移学习，输入分辨率统一为 $640 \times 640$，批次大小设为 32。训练过程持续 200 个 Epochs，采用 SGD 优化器并开启自动混合精度加速；在训练的最后 10 个 Epoch 强制关闭 Mosaic 数据增强，以消除坐标漂移并微调 BatchNorm 统计量。

3. 参数效率与实时性的平衡：YOLO11-Mamba 实现了轻量化设计，其参数量仅为 2.22 M，相比基准模型减少了约 15.3%。尽管计算量有所上升（17.85 GFLOPs），但得益于 SSM 的线性复杂度及硬件加速，推理速度仍高达 125.8 FPS，满足无人船端侧部署的实时性需求。
    

4.3 核心消融实验

4.3.1 物理模态的有效性验证

为了验证物理感知策略的有效性，本研究在固定网络结构的前提下，逐步引入不同模态策略。如表 2 所示，仅采用 RGB 模态在强反光场景下表现欠佳；引入偏振模态（DoLP）后精度提升显著，物理意义上是因为偏振度图像能有效增强漂浮物与水面背景的对比度；采用全模态随机训练策略后，模型达到最佳性能，证明学习跨模态不变特征能有效应对视觉伪装。

4.3.2 结构有效性分析

为验证 FCE 诊断出的语义塌陷问题，本研究设计了对比实验（见表 3）。实验结果显示，单纯翻倍骨干网络深度虽能略微提升特征抽象能力，但背景分离度提升有限且计算开销巨大。相比之下，在 P4/P5 引入 Mamba 模块，在仅增加少量参数的情况下取得了最佳的 FCE 指标。这证明 Mamba 的 SSM 机制能比单纯堆叠卷积层更有效地将前景从背景流形中剥离。

4.3.3 基于 FCE 诊断的层级选择验证

位置敏感性实验显示，将 Mamba 放置在浅层（P3）时性能提升微乎其微，因为浅层特征主要依赖局部纹理。而在 FCE 指引的 P4 和 P5 层级引入 Mamba 时，获得了最大性能增益，验证了 FCE 框架在架构优化中的指导价值。
